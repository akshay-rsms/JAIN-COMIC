# JAIN ONLINE — Comic Story Landing Pages

AI-generated, comic / Webtoon-style story landing pages, one per **search intent**, for
JAIN Online degrees. One locked HTML/CSS template (`build.py`) driven by per-story JSON, with
character-consistent art generated by Nano Banana (Gemini image model).

## The BBA set (8 pages, demand-driven)

Each page targets a real BBA search query, weighted by search volume.

| Story JSON | Target query | Persona | Intent |
|------------|-------------|---------|--------|
| `bba-course` | `bba course` | Ishaan | Generic / overview |
| `bba-fees-reveal` | `jain university bba fees structure` | Aarav | Fees (JAIN) |
| `bba-course-fees` | `bba course fees` | Sneha | Fees (comparison) |
| `bba-online` | `online bba course` | Aditya | Online / flexibility |
| `bba-online-home` | `bba online course` | Riya | Online / study from home |
| `bba-admission` | `bba admission` | Neha | Admission / how to apply |
| `bba-finance` | `bba in finance` | Rohan | Specialisation: Finance |
| `bba-digital-marketing` | `bba in digital marketing` | Kabir | Specialisation: Digital Mktg |

> Courses must match the real AMS/UMS grid. The BBA electives are Finance, Marketing, HR,
> Digital Marketing, Data Science & AI. "BBA in Aviation" had high search demand but is **not
> offered**, so it was dropped in favour of BBA in Finance.

## Template (7 chapters)
Origin · Setting the Target · Program Universe · Special Power · Character Evolution ·
Meet Sensei · Final Showdown. Cream `#fdf3da` / navy `#1a237e` / gold `#f5b301` palette,
Bangers display font, fully responsive (mobile: borderless full-width chapters, single-line
buttons, collapsible "show all" chip groups).

## Structure
```
build.py                 the comic template engine (JSON -> self-contained .html)
make_bba_stories.py      writes the 6 demand-driven BBA story JSONs
stories/*.json           storyboards (hero, 4 panels, lesson, proof, cta)
gen_bba_art.py           Nano Banana art: master -> face ref -> hero + 4 panels (no baked text)
gen_art.py               original art generator (master -> hero -> panels)
pages/*.html             built landing pages
pages/img/*              generated art (used by the pages)
share/*.html             self-contained shareable builds (fonts + images inlined)
```

## Build
```
py -3.12 make_bba_stories.py     # (re)write the demand-driven BBA story JSONs
py -3.12 gen_ratio_art.py        # generate art at placeholder ratios  (needs .env.local key)
py -3.12 build.py                # build all stories -> pages/
py -3.12 build.py bba-finance    # build one
py -3.12 pack_all.py             # self-contained share/*.html
```

## Art & consistency
`gen_ratio_art.py` reuses each story's master character sheet, crops it to the face, and passes
that as the reference for the hero (single character) and every panel, so the protagonist stays
consistent. Images are generated at the **placeholder aspect ratios** (hero 4:3, panels 3:2) and
cropped to exactly that ratio, so `background-size:cover` fills each box with no awkward crop.
Prompts forbid baked-in text and keep screens blank/glowing (Nano Banana misspells), staying
brand-safe and Webtoon-styled.

## Content standards (client checklist)
- Clear, correct English; brand-safe language, topics and imagery for a student audience.
- Hero/banner must be directly relevant to the page and consistent with its theme.
- Every story: complete beginning-middle-end across the 4 panels, no unresolved plot.
- No em/en dashes anywhere (`build.py` `sanitize()` enforces this).

## ⚠️ Verify before publishing
Stat-bar figures, fee/EMI/discount claims and testimonial names are illustrative in the
storyboard copy. Replace with verified JAIN Online numbers and real, consented testimonials
before any public launch. Aviation fees are intentionally left non-numeric (not confirmed).
